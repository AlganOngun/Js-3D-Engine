# Js-3D-Engine
3D Rendering Engine made from scratch using javascript, p5.js and electron.js

# Setup

# Prerequiestries
* electron.js 
```sh
npm install electron 
```
* nodejs and npm
```sh
sudo pacman -S nodejs
```

# Running
1) run the following command from commandline in the root of the repo:
```sh
npm start
```
## You're Done
